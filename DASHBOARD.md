## 🔑 Env Vars Needed on Vercel
| Variable | Where to Get It |
|---|---|
| `VAPI_PRIVATE_KEY` | Vapi Dashboard → Settings → API Keys |
| `NEXT_PUBLIC_VAPI_PUBLIC_KEY` | Vapi Dashboard → Settings → API Keys |
| `NEXT_PUBLIC_VAPI_ASSISTANT_ID` | Vapi Dashboard → Assistants → Copy ID |
| `NEXT_PUBLIC_CALENDLY_URL` | Your Calendly link |
| `CALENDLY_LINK` | Same Calendly link |
| `TWILIO_ACCOUNT_SID` | Twilio Console (when you add SMS) |
| `TWILIO_AUTH_TOKEN` | Twilio Console |
| `TWILIO_PHONE_NUMBER` | Twilio Console |
